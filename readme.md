## Installation 
- Clone the repo

  git clone https://github.com/AshleighWilliams/first-nextjs-app.git
 
- Install NPM Packages
 
-To run this code:
1. cd to project folder
2. mkdir .next
3. npm install
4. npm run dev
5. http://localhost:3000/
